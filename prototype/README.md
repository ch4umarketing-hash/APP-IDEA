# Prototype usage

## Run locally
From the repository root:

```bash
python -m http.server 8000
```

Then open:

- `http://localhost:8000/prototype/`

## What you can interact with
- Login screen (demo auth)
- Dashboard KPIs
- Add stock
- Sell/remove stock
- Transfer stock between stores
- Create customer repair tickets

Data is persisted in browser `localStorage` so you can refresh and continue testing.
